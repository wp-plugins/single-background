== Installation ==

### Installation

1. Upload 'single-background' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

### Shortcode

1. [single_bg url="" color=""]

### [single_bg] Attributes

1. url="paste here your background image url".
2. color="paste here color code or word, example for hex code: #000000 or enter black, example for rgba code: rgba(0, 0, 0, 1)".

### Example

	[single_bg url="http://qass.im/book.jpg"] this is for background image.
	[single_bg color="#000000 or black or rgba(0, 0, 0, 1)"] this is for background color.
